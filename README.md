- Basics of digital image processing

### Week 2 – Neural Networks & Image Processing
**Focus:**
- Neural network fundamentals and intuition
- Image processing using Python
- Medical imaging dataset exploration

**Key Tasks Implemented:**
- Grayscale conversion
- Histogram visualization
- Image resizing
- Edge detection
- Noise addition and Gaussian noise removal
- Exploration of medical imaging datasets (X-ray, MRI, etc.)
- Analysis of dataset size, classes, imbalance, and challenges

**Deliverables:**
- Python/Jupyter notebooks
- Dataset summaries
- Visual outputs and plots

---

### Week 3 – CNNs, Darknet, and YOLO
**Focus:**
- Convolutional Neural Networks (CNNs)
- ImageNet fundamentals
- Darknet framework
- YOLO (You Only Look Once) using Ultralytics

**Key Work:**
- Study of popular CNN architectures (AlexNet, VGG, ResNet, EfficientNet)
- Dataset exploration using Roboflow
- Training a classification model using YOLOv8 / YOLOv11
- Model evaluation and result analysis

**Deliverables:**
- Training notebook (run on Kaggle / Colab)
- Results PDF including:
  - Dataset overview
  - Model used
  - Training accuracy and loss
  - Sample predictions
  - Observations and limitations
